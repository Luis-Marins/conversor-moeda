# Conversor de Moeda (Real ↔ Dólar)

Programa em Java que converte valores entre Real (BRL) e Dólar (USD) utilizando uma taxa de câmbio fixa.

## Como executar

1. Compile: `javac ConversorMoeda.java` ou execute direto com `java ConversorMoeda.java`
2. Execute: `java ConversorMoeda`
3. Selecione `1` para converter de Real para Dólar ou `2` para Dólar para Real e insira o valor quando solicitado.
